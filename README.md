# TODO-Applications
 To-Do List Application is a console-based program developed using Python to manage daily tasks. It allows users to add, view, and remove tasks through a simple menu-driven interface. The program uses lists, loops, and conditions for functionality. It helps beginners understand basic programming concepts and build a practical task management system

A simple, console-based Python application to manage your daily tasks.

 FEATURES:
1. Add new tasks to a list.
2. View all current tasks.
3.Remove completed tasks.
4.User-friendly menu interface.
